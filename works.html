Mapping Feature for a Web App used by Deconstruction Professionnal

This is a feature I conceived and implemented during my fullstack intership at Afleya on their application Akibo.


Akibo is a web application used in decontruction project (renovation or demolition of buildings). It provides amongst other things tools to list available materials on site and sort them between recyclable, reusable and waste materials. I conceived and implemented a feature in the existing application allowing to upload maps of the buildings and to locate the various materials on them through a drag and drop interface. One of the main challenges was that the feature had to work offline.


My contract means the code I have written belongs to Afleya but my employer acknowledged me on linkedin for it.

CNN applied to Circuit Fault Diagnosis

This is a student research project inside the CNRS laboratories on campus: LIRIS (Computer Science) and INL (nanotechnology).


In this project, my classmate and I experimented with the application of Convolutional Neural Network to the diagnosis of fault in electronic circuits. The main challenge is that faults are not always "permanent". Applying the same input multiple times to a circuit with a unique "intermittent" fault can give different outputs.

Using a sample of the numerous outputs possible for a single type of "intermittent" fault, we were able to train a neural network to recognize said "intermittent" fault without having to list all cases.

This allowed us to propose and test a hierarchical approach where we use a "tree" of networks to successively distinguish between a few groups of faults until there is only one left. We did not implement the whole "tree", only 2 layers but the results were promising.


The report is available here at the school library (Bibliothèque Michel Serres).

Autonomous UAV Flight for Delivery

This is a student project in collaboration with the student association Centrale Lyon Cosmos and the CNRS laboratory on campus: LIRIS (Computer Science).


In this project, my classmates and I attempted to modify a UAV kit to make an autonomous delivery drone. The project did not come to fruition but we learned tremendously about electronics, flight mecanics, control engineering, computer vision and pathfinding algorithms.

Simulation of a Pedestrian Evacuation

This is the project I presented for my "TIPE" on the theme "Health, Prevention" in 2021. The "TIPE" is an exam that is part of the nationwide entrance examination for France's Engineering Schools.


In this project, I implemented a simplified version of Helbing's model to simulate the evacuation of a 2D crowd of pedestrian through a door. I also filmed small scale experiments with volunteers and replicated the situations with my simulation to compare the model and reality.
